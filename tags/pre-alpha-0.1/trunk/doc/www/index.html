Efficient XML Interchange Processor

Project details

  • Summary - EXI Processor (EXIP) is a project started at EISLAB, Lule√• University of Technology. It is part of research efforts to bring resource-constrained embedded devices, such as Wireless Sensor Nodes, closer to the enterprise business processes taking place in manufacturing and communication industries.
  • Motivation - As Efficient XML Interchange data format is heading towards becoming a W3C standard, the only open source implementation of parser available is EXIficient. It is written in Java, not quite mature yet and depends on XML parser libraries. This makes it inappropriate for resource-constrained embedded systems development where the EXI format is expected to make big impact similarly to how XML standard is now used in almost all enterprise applications.
  • Project stage - System design
