Change log
Version 0.5.3 (Bugfix alpha release)
  • Bug fixes
  • Partial support for the EXI Profile specification
Features:
  • EXI Cookie
  • Options Presence Bit
  • Strict mode
  • Fragment
  • Preserve.prefixes
  • SchemaId
  • ValueMaxLength
  • ValuePartitionCapacity
  • Schema parser for EXI encoded XML Schemas
Known issues:
  • Compression and pre-compression are not supported
  • Preserving lexicalValues is not supported
  • Random access through selfContained option is not supported
  • Datatype Representation Map is not supported
  • Some exotic XML schema constructs are not implemented yet (e.g. string pattern facets for Restricted Character Sets, Substitution groups)
